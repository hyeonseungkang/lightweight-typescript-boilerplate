# Lightweight Typescript Boilerplate

## Description

- Typescript boilerplate for tiny codebase, office-work automation.
- Instant starts with swc, eslint, prettier, jest.

## Project setup

```bash
$ npm --include=dev install 
```

## Write code

- *src/main.ts*
    - `npm start` starts here.
- *.spec.ts
    - Jest test codes.

## Compile and run the project

```bash
# build
$ npm run build

# watch mode
$ npm run start:dev

# production mode
$ npm run start
```

## Run tests

```bash
# unit tests
$ npm run test

# watch mode
$ npm run test:watch

# test coverage
$ npm run test:cov
```
